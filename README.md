- Arrays
- Binary Trees & Binary Search Trees
- HashMaps
- Heaps (Priority Queues)
- Kadane's Algorithm
- Queues
- Stacks
- String Manipulation
- Bit Manipulation
- Sliding Window & Two Pointers
